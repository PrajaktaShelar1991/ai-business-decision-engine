# Case Study: AI Business Decision Engine

## Problem
Decision-making in organizations is slow and fragmented.

## Goal
Build a system that converts business problems into actionable outputs.

## Approach
Designed a multi-agent workflow:
- Problem understanding
- Data analysis
- Insight generation
- Strategy creation
- Execution planning

## Demo
Input: Revenue dropped by 15%

Output:
- Identified mobile performance issues
- Suggested optimization strategy
- Generated PRD and roadmap

## Outcome
Simulated real-world PM workflow using AI agents.

## Learnings
- Structured prompts improve results
- Breaking problems into agents improves clarity
